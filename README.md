# Boost-Macro-Updates

## 📖 Overview

Boost-Macro-Updates provides the **best Boost Macro in the game, totally free**! This repository offers a powerful, easy-to-use macro solution designed to enhance your boosting experience by automating repetitive actions and giving you that competitive edge.

## ✨ Features

- **Instant Performance Boost**: Experience immediate improvements in your gameplay with a configurable macro designed for optimal performance.
- **Totally Free**: No subscriptions, no hidden costs—just pure, unrestricted access to all features.
- **Easy Setup**: Quick installation process with files (alt.zip and main.zip) that get you up and running in minutes.

## 🚀 Installation

1. **Download the Macro:**
   - Download the ZIP directly: Click the green "Code" button → "Download ZIP"

2. **Extract the Files:**
   - You'll find two archive files in the root directory: `main.zip` and `alt.zip`
   - Choose the appropriate version based on what you need and extract it:
     - **main.zip**: The Macro Used On Your Main Account During Boosts
     - **alt.zip**: The macro to use on your alts during boosts

## 🎮 Usage

1. **Launch Your Game:**
   - Start the game when you want to use the boost macro

2. **Adjust Settings:**
   - Modify the configuration files to customize what you need

3. **Start Boosting:**
   - Activate the macro using your configured hotkey
   - Watch your boost numbers improve
